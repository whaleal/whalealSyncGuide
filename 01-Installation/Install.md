# Install

    MongoT下载安装与配置

<br>

## 下载MongoT

a.访问 https://github.com/whaleal/mongodbT/releases

b.下载最近版本的mongodbT.tar.gz


<br>



## 解压缩

a.新建目录 mongodbT

    mkdir mongodbT

b.解压至mongodbT中

    tar -zxvf mngodbT.tar.gz  -C mongodbT




## 配置文件修改

a.切换目录

    cd mongodbT/config

b.编辑配置文件

    vi mongodbT.properties


Properties文件[配置详情](../02-Connecting/MongodbTConfig.md)
