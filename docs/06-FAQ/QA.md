# 常见问题与解答

## 能否创建索引？

    能

## 能否多任务并行执行？

    不可以


## 建表时表的复杂结构会同步吗？

    不支持视图 不支持TimeSeriesCollection。


## 支持哪些集群间的同步




|   S/T   |单节点  | 复制集   |分片  |
|  :----:  | :----:  |:----: | :----: |
| 单节点  | 全量 |全量 |全量 |
| 复制集  | 全量,实时,全量加增量,全量加实时 | 全量,实时,全量加增量,全量加实时 | 全量,实时,全量加增量,全量加实时 |
| 分片  | 全量 | 全量 | 全量 |


复制集对5.0是实时同步不支持
